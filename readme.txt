WPC Linked Variation is built to link separate products together by attributes. Extended with custom album classifications.

== Description ==

本插件基于 **WPC Linked Variation for WooCommerce** 官方 4.4.2 免费版本进行二次开发与功能增强。

> 特别感谢 **wpclever** 团队提供如此优秀的开源基础。

### 🚀 修改版核心增强特性：
*   **默认添加专辑分类 (Custom product_collection Taxonomy)：** 自动为产品引入全新的“专辑/系列”分类管理维度。
*   **支持按term关联产品 (term Linking)：

== Installation ==

1. 将 `wpc-linked-variation` 文件夹上传到 `/wp-content/plugins/` 目录，或者直接在后台上传插件 `.zip` 压缩包。
2. 在 WordPress 插件管理后台激活它。
3. 前往 WooCommerce 产品管理，你会发现新增了“专辑”分类，并在 WPC 关联设置中支持按专辑批量联动。
